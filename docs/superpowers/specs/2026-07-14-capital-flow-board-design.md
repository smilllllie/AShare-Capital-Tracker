# A股资金流向可视化系统设计文档

## 1. 目标与范围
构建一个极具视觉冲击力的 A 股资金流向动态可视化工具。该工具需同时满足：
- 浏览器内交互式的动态加载动效（霓虹发光质感、数值滚动）。
- 每天通过 Python 自动获取真实市场数据并生成静态截图，推送到手机端，截图必须100%保留网页的高级UI质感。

## 2. 架构设计

### 2.1 视觉展现层 (HTML/CSS/JS)
- **文件**：`capital_flow_template.html`
- **配色系统**：
  - 背景：极致暗黑/深蓝灰 (`#0F172A` 或更深)。
  - 流入（看多）：赤红色系渐变加外发光。
  - 流出（看空）：翠绿色系渐变加外发光。
- **布局设计**：
  - 顶部：日期、A股大盘核心指数（如上证指数）。
  - 中部：净流出 Top 10 榜单（绿色）。
  - 底部：净流入 Top 10 榜单（红色）。
- **动态渲染与比例同步**：
  - 数据注入格式：网页将预留一个全局的 JSON 变量 `window.CAPITAL_DATA`，由 Python 填入真实数据。
  - 比例计算：前端读取数据后，自动找出流入和流出的“最大绝对值”。其他所有板块的进度条长度严格按照 `(当前值 / 最大值) * 100%` 的比例渲染，确保递增和真实数据绝对同步。

### 2.2 数据采集与处理层 (Python)
- **文件**：`generate_board.py`
- **核心库**：`akshare`, `pandas`
- **流程**：
  1. 调用 `akshare` 接口（如 `stock_sector_fund_flow_rank` 等）获取今日板块资金流向。
  2. 清洗数据，按净额排序，分离出流入前10和流出前10。
  3. 将数据格式化为 JSON 字符串。
  4. 读取 `capital_flow_template.html` 源码，用真实数据 JSON 替换占位符，生成一份带有数据的 `capital_flow_render.html`。

### 2.3 自动化截图与推送层 (Python)
- **核心库**：`playwright`
- **流程**：
  1. 启动 Playwright 无头浏览器。
  2. 加载 `capital_flow_render.html`。
  3. 注入延迟等待（例如 1.5 秒），确保进度条伸展和数值滚动的动画执行完毕。
  4. 对指定 DOM 节点或全页截取全尺寸、高清长图（`.png`）。
  5. 调用现有推送通道（复用当前目录下的推送逻辑）将图片发送至手机。

## 3. 约束与边界
- 确保浏览器截图的环境兼容性（仅需安装 `playwright` 及其浏览器依赖）。
- 网页样式使用纯内联/本地相对路径资源，确保本地直接打开 `.html` 文件即可完美展示，不依赖外部服务器。
